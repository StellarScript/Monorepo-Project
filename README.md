## Doppler Usage

-  doppler login

```bash
dopper login
```

<br>

-  run any command with dopper

```bash
doppler setup --no-interactive -p appify -c prod && <COMMAND>
```

<br>

-  example command

```bash
doppler setup --no-interactive -p appify -c prod && nx deploy cdk --stack service
```
